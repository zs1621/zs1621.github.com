  • 如果需要压缩的文件 符合过滤条件 也即在给的文件列表中 就继续下一步, 否则 return
  • 如果请求头文件 没有 accept-encoding 值 ， return (why)
  • 如果请求方法 为 HEAD , return
  • 默认压缩方法为 gzip
  • 确定压缩方法 即给 method 赋值
  • 此时method 还是空的话 return
